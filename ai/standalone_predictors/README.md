# Standalone Predictors

Thư mục này gom các predictor VIP riêng để tránh treo ở thư mục gốc:

- `loto_5_35_predictor/`
- `mega_6_45_predictor/`
- `power_6_55_predictor/`
- `predictor_v2/` là predictor legacy giữ lại làm fallback

Mỗi predictor có `config/`, `data/`, `models/`, `state/`, `src/`, `tests/` riêng.
